# PowerShell Zombie Survival Game

## Description
This is a beginner text-based zombie survival game built in PowerShell as part of a classroom project. Players progress through the story by making decisions that determine the outcome of the game. The project was created to practice basic PowerShell scripting concepts while learning how to build interactive command-line applications.

## Features
- Decision-based gameplay
- Multiple story paths
- User input handling
- Basic game logic
- Interactive command-line interface

## Technologies Used
- PowerShell

## Skills Demonstrated
- Variables
- Functions
- Conditional statements
- Loops
- User input
- Basic scripting logic

## How to Run
1. Clone or download this repository.
2. Open PowerShell.
3. Navigate to the project folder.
4. Run the script:

```powershell
.\ZombieGame.ps1
```
